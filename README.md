# Shanna Rebekah Portfolio Website

Top quality A* A-level coursework

## How to run
Requires python 3.6 or above

### Linux
```
python3 -m venv venv
. venv/bin/activate
pip install -r requirements.txt
python app.py
```

### Windows
```
py -3 -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
python app.py
```
